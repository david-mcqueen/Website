Testing Application

(Knockout App)

Developed in my spare time, this browser based application was created to stramline the existing testing process at my employer, Daviker. The application allows for comprehensive regression testing, providing realtime testing progress & status to be provided to the team when we are conducting our pre-release testing. The test results are aggregated, allowing for multiple people to provide feedback on the same area, and presented in a clean concise summary.

The development of this application helped to streamline the testing process, and reduce the amount of manual collation & processing of the test results, allowing for more time to be spent testing and fixing any found bugs.

This application was built with a KnockoutJS frontend, with a .net core and PostgreSQL backend.

Pulser

(iPhone App)

Pulser is an audible heart rate monitor, connecting directly to your Bluetooth (BLE) device to provide you constant updates as to how hard you are pushing your heart. Know when you are at your peak, and know when you can give that little bit extra and set a new personal record. Hear your heart, and reach your training goals.

Pulser was created after noticing that some activity trackers failed to provide realtime audible data on the users heart rate, even though it was being tracked. Pulser is written natively in Swift, and integrates seamlesly with activity trackers to achieve the best of both worlds.

I no longer publish Pulser to the app store, however Pulser is an open source project allowing users to fork the project and create the own incarnation of the app.
